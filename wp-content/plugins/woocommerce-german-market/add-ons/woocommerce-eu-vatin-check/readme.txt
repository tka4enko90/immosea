== Description ==

Adds a field for value-added tax identitification number (VATIN) during checkout. Validates field entries against the official web site of the [European Commission](http://ec.europa.eu/taxation_customs/vies/vieshome.do?locale=en).

== Installation ==

1. If you don’t know how to install a plugin for WordPress, [here is how](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins).
2. In your WordPress admin go to WooCommerce → Options → VATIN Check and configure your settings.
